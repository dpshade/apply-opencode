## Features

- Generate 500-1000 characters at cursor position
- Select text to replace it with AI-generated content
- Uses note title, frontmatter, and surrounding context
- Optional instruction to guide generation

## Usage

1. Place cursor where you want content, or select text to replace
2. Run: **Apply OpenCode: Generate content at cursor**
3. Optionally type an instruction (e.g., "summarize", "expand", "add examples")
4. Press **Enter** to generate

<Tip>
Leave the instruction empty for natural continuation based on context.
</Tip>
