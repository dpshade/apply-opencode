## Features

- Generate meaningful 2-5 word titles from note content
- Ribbon icon (brain) for quick access
- Optional confirmation modal before rename
- Bulk rename all "Untitled" files at once

## Usage

1. Open a note
2. Click the brain icon in ribbon, or run: **Apply OpenCode: Generate AI title for current file**
3. File is renamed based on content

## Bulk Rename

In plugin settings, click **Bulk rename untitled files** to rename all notes containing "Untitled" in the filename.
