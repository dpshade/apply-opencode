## Requirements

- [OpenCode CLI](https://opencode.ai) installed and configured
- Default path: `~/.opencode/bin/opencode`

## Using BRAT (Recommended)

<Steps>
  <Step title="Install BRAT">
    Install [BRAT](https://github.com/TfTHacker/obsidian42-brat) from Community Plugins
  </Step>
  <Step title="Add Plugin">
    Open BRAT settings → "Add Beta plugin" → Enter `dpshade/apply-opencode`
  </Step>
  <Step title="Enable">
    Enable in Settings → Community plugins
  </Step>
</Steps>

## Manual Installation

1. Download `main.js`, `manifest.json`, and `styles.css` from the [latest release](https://github.com/dpshade/apply-opencode/releases)
2. Create folder `.obsidian/plugins/apply-opencode/` in your vault
3. Copy downloaded files into the folder
4. Reload Obsidian and enable the plugin
