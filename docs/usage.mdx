## Commands

| Command | Description |
|---------|-------------|
| **Enhance note frontmatter** | Analyze content and suggest frontmatter |
| **Generate AI title** | Create meaningful title from content |
| **Generate content at cursor** | AI-generate content at cursor position |
| **Identify and add wiki links** | Find entities that should be linked |

## Settings

| Setting | Description |
|---------|-------------|
| Model | OpenCode model for AI operations |
| Executable path | Path to opencode CLI |
| Diff view style | Side-by-side or unified diff |
| Max list items | Limit for array properties |
| Wiki link strategy | Existing notes only or all entities |
| Wiki link mode | First mention only or all mentions |
| Confirm before rename | Show modal before title rename |

## Bulk Operations

Available in plugin settings:

- **Bulk rename untitled files** - Rename all files with "Untitled" in name
- **Bulk enhance frontmatter** - Add frontmatter to files missing it
