# Apply OpenCode

AI-powered note enhancement for Obsidian using [OpenCode](https://opencode.ai) CLI.

<CardGroup cols={2}>
  <Card title="Frontmatter Enhancement" icon="tags" href="/features/frontmatter">
    Analyzes content and suggests frontmatter based on similar notes in your vault
  </Card>
  <Card title="AI Titles" icon="heading" href="/features/titles">
    Generate meaningful 2-5 word titles from note content
  </Card>
  <Card title="Content Generation" icon="wand-magic-sparkles" href="/features/content">
    Generate content at cursor or replace selections with AI
  </Card>
  <Card title="Wiki Links" icon="link" href="/features/wiki-links">
    Identify entities that should become wiki links
  </Card>
</CardGroup>

## Quick Start

<Steps>
  <Step title="Install OpenCode CLI">
    Follow instructions at [opencode.ai](https://opencode.ai)
  </Step>
  <Step title="Install Plugin">
    Use BRAT with repo `dpshade/apply-opencode`
  </Step>
  <Step title="Enhance a Note">
    Open a note and run **Apply OpenCode: Enhance note frontmatter**
  </Step>
</Steps>
